Vehicle Insurance Fraud

Visualizing trends and Detecting Fraudulent Claims

Fighting Fraud with Machine Learning

Fraud is an increasingly expensive and widespread problem for insurers, and fraud detection has historically required time-consuming manual review. This project attempts to use Logistic Regression to automate this process.

Visualizing with Tableau

We used Tableau to explore our dataset, comparing fraud detection by month, by sex of claimant (ADD MORE). This analysis revealed that men and women have the same rough likelihood of submitting fraudulent claims. We found that the number of fraudulent claims trends downward month over month throughout the year.
